## 确认主板IO型号
- 确认 X99 主板的 Super I/O 芯片型号
	~~~shell
	sudo apt install lm-sensors
	sudo sensors-detect
	~~~
	执行后，在所有输入的地方输入YES
	~~~shell

	# System: YiChenYu X99M-Gaming [Default string]
	~~~

## 加载内核模块
- 使用`lsmod`确认模块是否加载
	~~~shell
	lsmod | grep nct6775
	~~~
	如果没有输出，则加载对应模块：
	~~~shell
	sudo modprobe nct6775
	~~~
	确认加载完成：
	~~~shell
	lele@lele-X99M-Gaming:~$ lsmod | grep nct6775
	nct6775                40960  0
	nct6775_core          106496  1 nct6775
	hwmon_vid              12288  1 nct6775
	~~~

## 手动控制风扇
- 启用手动控制
	~~~shell
	echo 1 | sudo tee /sys/class/hwmon/hwmon3/pwm2_enable
	~~~
- 转速拉满(0最低，255最高)
	~~~shell
	echo 255 | sudo tee /sys/class/hwmon/hwmon3/pwm2
	~~~
	这样就可以看到风扇呼呼转了，声音贼大
