## 编译llama.cpp
- 安装必要依赖
	~~~shell
	sudo apt install build-essential
	sudo apt install cmake git libcurl4-openssl-dev libstdc++-12-dev
	~~~
- 下载源代码 
  [官网](https://github.com/ggml-org/llama.cpp/releases)
  ![在这里插入图片描述](./tinkering/13.png)
  在release界面选择最新版本的源码
- 解压
	~~~shell
	tar -xvf llama.cpp-master.tar.gz
	cd llama.cpp-master
	~~~
- 使用`cmake`
	~~~shell
	HIPCXX="$(hipconfig -l)/clang" HIP_PATH="$(hipconfig -R)" \
	    cmake -S . -B build -DGGML_HIP=ON -DAMDGPU_TARGETS=gfx906 -DCMAKE_BUILD_TYPE=Release
	~~~
	官网的编译实例使用的是gfx1030，我们到[这个网站](https://packages.gentoo.org/useflags/amdgpu_targets_gfx1030)看mi50对应的是什么
	![在这里插入图片描述](./tinkering/14.png)
- `cmake`成功之后直接编译即可
	~~~shell
	cmake --build build --config Release -- -j 16
	~~~

## 运行Qwen3系列
### 4B Q4
- 目录结构如下
	~~~shell
	lele@lele-X99M-Gaming:~$ tree -L 1
	.
	├── amdgpu-install_6.0.60001-1_all.deb
	├── llama.cpp-master
	└── Qwen3-4B-UD-Q4_K_XL.gguf
	~~~
- 执行命令（注意要使用sudo，不然可能会识别不到gpu）
	~~~shell
	sudo ./llama.cpp-master/build/bin/llama-cli -m Qwen3-4B-UD-Q4_K_XL.gguf -ngl 99
	~~~
