## 安装wails
- 设置go国内代理
	~~~bash
	go env -w GOPROXY=https://goproxy.cn
	~~~
- 安装
	~~~bash
	go install github.com/wailsapp/wails/v2/cmd/wails@latest
	~~~
- 确认安装

## 创建项目
- 以前端框架为`vue`+`typescript`为例，创建名为`wailsdemo`的项目：
	~~~bash
	wails init -n wailsdemo -t vue-ts
	~~~

## 运行
- 直接执行命令
	~~~shell
	wails dev
	~~~
	![在这里插入图片描述](./golang/9.png)
- 编译
	~~~bash
	wails build
	~~~
	默认项目编译后exe文件大小在10MB左右
	![在这里插入图片描述](./golang/10.png)
在另一台机器运行时，除webview之外的内存占用在5MB左右
	![在这里插入图片描述](./golang/11.png)
	虽然说进程本身的占用率不高，但是webview的内存占用量不低
