## 使用docker启动
- 拉取docker镜像
```shell
docker pull agent0ai/agent-zero:latest
```

国内镜像加速(下面的镜像加速可能会过期，建议找一个镜像网站加速)
```shell
docker pull swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/agent0ai/agent-zero:latest
```

- 拉取完成之后可以启用(注意不要使用80端口，否则会报错)
```shell
docker run -p 8080:80 swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/agent0ai/agent-zero:latest
```

- 或者在docker desktop中启用
